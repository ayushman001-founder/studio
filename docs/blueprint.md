# **App Name**: ConviCo: Your AI Companion

## Core Features:

- User Authentication: Secure login/registration via mobile number or email.
- Profile Creation: Users can customize their profile with age, gender, language (English, Hindi, Hinglish), and country (India).
- AI Character Selection: Users choose from categories (Love, Family, Friend, Business), each featuring AI companions (e.g., Dad, Mom, Boyfriend, Girlfriend, Mentor, Co-founder). Each character uses a realistic, non-cartoon display picture.
- AI Chat Interface: Engage in conversations with AI companions that provide short, friendly, and natural-sounding responses. This tool employs advanced AI to understand the context and offer tailored advice and support.
- Contextual AI Responses: AI companions generate responses tailored to the selected category (Love, Family, Friend, Business), providing relevant support and conversation.
- Character Customization: An AI powered feature that lets you fine-tune your Ai friend, changing his or her behavior based on a list of options.

## Style Guidelines:

- Primary color: Calm blue (#72BCD4) to inspire trust and companionship.
- Background color: Soft light gray (#F0F4F7) for a clean and approachable feel.
- Accent color: Warm coral (#FF8A65) to highlight important elements.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Use simple, outlined icons to represent different categories and actions, in a friendly and approachable style.
- Maintain a clean, card-based layout with clear spacing for easy navigation.
